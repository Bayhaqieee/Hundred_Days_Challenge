HTML & CSS Basics Summary

  1. HTML Summary
  2. HTML (HyperText Markup Language) is used to define our page content, structure and meaning. You don't use it for styling purposes. Use CSS for that instead!
  3. HTML uses "elements" to describe (annotate) content
  4. HTML elements typically have an opening tag, content and then a closing tag
  5. You can also have a void(empty) elements like images
  6. You can also configure elements with attributes
  7. There's a long list of available elements but you'll gain experience over time, no worries
  8. Learn more about all available HTML elements on the MDN HTML element reference
  9. CSS Summary
  10. CSS (Cascading Style Sheets) is used for styling your page content.

  11. Styles are assigned via property-value pairs
  12. You can assign styles via the "style" attribute
  13. To avoid code duplication, you typically use global styles (e.g via the "style" element) instead
  14. Alternatively, you can work with external stylesheet files which you "link" to
  15. When working with CSS, concepts like "inheritance","specifity" and the "box model" should be understood
  16. Learn more about all available CSS properties and values on the MDN CSS property reference
